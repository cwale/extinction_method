# Extinction Timer

A single-page web app for the Hiscock graduated extinction ("controlled comforting") protocol. Designed for exhausted parents standing in a dark nursery at 3 am.

## How to run

Open `index.html` directly in a browser:

```
open index.html
```

Or serve locally (required for Wake Lock API on some browsers):

```
python3 -m http.server
```

Then open `http://localhost:8000` in your browser.

No build step. No dependencies. Works offline after first load.

## Note

This tool supports a behavioural sleep protocol commonly recommended by paediatricians and sleep specialists. It is not medical advice. Use it as directed by your own health professional, who can assess whether graduated extinction is appropriate for your child's age, temperament, and circumstances.
